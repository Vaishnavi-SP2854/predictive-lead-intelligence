# 🚀 Predictive Lead Intelligence

## 📌 Project Overview

Predictive Lead Intelligence is a machine learning project focused on analyzing marketing campaign data to identify high-quality leads and improve conversion efficiency.

The objective is to help businesses:

- Identify where leads drop off in the funnel
- Detect high-performing marketing channels
- Predict customer conversion probability
- Optimize campaign strategy using data-driven insights

---

## 📊 Dataset

Dataset Used: Bank Marketing Dataset  
Target Variable: `deposit`

- `1` → Customer Converted
- `0` → Not Converted

Each row represents a contacted lead during a marketing campaign.

---

## 🔍 Project Phases

### 🟢 Period 1 – Data Understanding & EDA
- Explored dataset structure
- Analyzed conversion distribution
- Evaluated contact channel performance
- Identified funnel drop-off patterns

### 🟢 Period 2 – Data Cleaning & Feature Engineering
- Encoded target variable
- Applied one-hot encoding
- Prepared ML-ready dataset
- Created train-test split

### 🟢 Period 3 – Model Building
- Trained Logistic Regression
- Trained Random Forest
- Evaluated using Accuracy, Precision, Recall, F1, ROC-AUC
- Compared model performance

### 🟢 Period 4 – Business Insights
- Identified key conversion drivers
- Analyzed channel-level effectiveness
- Provided marketing optimization recommendations
- Detected data leakage (call duration variable)

---

## 🤖 Models Used

- Logistic Regression
- Random Forest Classifier

Evaluation Metrics:
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

---

## 💼 Key Business Insights

- Cellular contact method shows highest conversion rate.
- Unknown contact channel shows significant drop-off.
- Financial attributes influence conversion probability.
- Predictive lead scoring can significantly improve marketing ROI.

---

## 📈 Business Impact

By implementing predictive lead scoring:

- Improve campaign targeting
- Reduce outreach costs
- Increase conversion efficiency
- Optimize marketing budget allocation
